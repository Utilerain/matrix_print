You can do a type of quest with a chat or something else... All according to your desire!


2.Commands

The first thing we need to import the matrix_print

1|import matrix_print as mp

Then you can get started!

1| mp. printc (text, time) #classic printing
2| mp. printu (text, time, name_of_user) #by the type of message sent from the user
3| mp.matrix_input(text, time, your_name="") #same input, but with the print effect

There is also a special class- ColorPrint

If you don't like writing mp. ColorPrint. x, you can just import the class

1|from matrix_print import ColorPrint as col

if the library does not detect Colorama, it will output an error

matrix_print.matrix.ColoramaIsNotInstalledError: module colorama is not installed

But if you have installed this module, then you are well done!

1|col.red(text, time) # print red text
2|col.bg_red(text, time) # print text with a red background
3|col.yellow(text, time) # printing with yellow text
4|col.bg_yellow(text, time) # print text with a yellow background
5|col.blue(text, time) # print with blue text
6|col.bg_blue(text, time) # print text with a blue background
7|col.green(text, time) # print with green text
8|col.bg_green(text, time) # print with a blue background
9|col.bg_white(text, time) # print with a white background

3.Example of using

import matrix_print as mp
from matrix_print import ColorPrint as col
import random
import time
import sys

mp.printc("hi, welcome to elOS", 0.1)
mp.printc("elastic Operating System", 0.1)
mp.printc("downloading files")
time.sleep(1)
wat = random.choice(["ERROR", "ok"])
if wat == "ERROR":
  col.red(wat, 0.1)
  time.sleep(1)
  sys.exit()
else:
  col.green(wat, 0.1)
